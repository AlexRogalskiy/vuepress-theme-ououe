# 介绍

基于 VuePress 的一个博客主题，部分设计参考 TryGhost/Casper

在线演示: My Blog

使用我准备好的模板 vuepress-theme-ououe-template 快速开始

# 特色

  • 浅色与深色主题自适应或者自定义默认主题颜色
  • 界面自适应电脑与手机
  • 图片懒加载包括 md 文件里面的图片
  • 支持多文件夹的博客
  • 一个筛选过的主页列表
  • 兼容 基础路径
