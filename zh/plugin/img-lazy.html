# vuepress-plugin-img-lazy ^1.3.7

提示

内置插件，此插件你可以直接使用或者修改插件配置

base on markdown-it-img-lazy and markdown-it-imsize and lozad

演示

源码

提示

一般情况下，你只需要了解如何在 markdown 文档下使用即可

![img](img.jpg)
<!-- 或者 -->
![img](img.jpg =500x300) <!-- 最佳 -->
<!-- 或者 -->
<template>
  <img-lazy src="img.jpg" />
<template>

# 修改配置

提示

在 1.4.0 以上的版本时，你可以同时控制 markdown 文档与主题的图片

module.exports = {
  plugins: [
    [
      'img-lazy',
      { /* 配置 */ }
    ]
  ]
}

# 配置

# useLoading

  • Type: Boolben
  • Default: true

是否使用基于原生的懒加载

提示

一般情况下使用原生的懒加载会比不使用加载更多的图片

# selector

  • Type: string
  • Default: lazy
