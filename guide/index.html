A blog theme for VuePress, Take TryGhost/Casper as a reference

Live Demo: My Blog

Uses vuepress-theme-ououe-template to starter

# Features

  • prefers-color-scheme or custom default theme color
  • sadaptive computer and mobile
  • image lazy-loading
  • support for multi folder blogs
  • a filtered list of home pages
  • compatible Base URL
