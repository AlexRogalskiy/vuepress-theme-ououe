# Options

# defaultTheme ^1.3.6

  • Type: string, object
  • Required: false

TIP

By default, light or dark themes are displayed by prefers-color-scheme, You can change it by defaultTheme

support light, dark or { theme: [begin hours, end hours] }




 

 

 




 





module.exports = {
  theme: 'ououe',
  themeConfig: {
    defaultTheme: 'dark',
    // or
    defaultTheme: { dark: [18, 6] },
    // or
    defaultTheme: { light: [6, 18], dark: [18, 6] },
  },
  // When using `defaultTheme`, you need to add a postcss plugins to your config.js
  postcss: {
    plugins: [
      require('css-prefers-color-scheme/postcss'),
      require('autoprefixer')
    ]
  }
}
